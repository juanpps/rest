# Documento Maestro de Diseño UX/UI

## Principios
- Soberbio, sobrio, editorial.
- Estética Romano-Vikinga sutil.
- Tipografía generosa y jerarquía clara.

## Tokens de Diseño
### Colores (Modo Claro)
- Background: `#f1e8d8` (pergamino)
- Text: `#1c1c1c`
- Accent: `#7a1c1c` (rojo imperial)
- Gold: `#b08d57`

### Colores (Modo Noche)
- Background: `#070707`
- Text: `#e8e6e0`
- Accent: `#5a1111`
- Gold: `#8c6e45`

### Tipografías
- Títulos: `Cinzel`
- Cuerpo: `Inter` / `Roboto`
- Citas: `Playfair Display`

## Componentes
- **Header:** Logo IM minimal + PIN access.
- **Sidebar:** Índice anual y filtros.
- **Editor:** Canvas central editorial.
- **Archive:** Grid de portadas minimalistas.
